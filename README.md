# Cloth Market — Gumti Square, Faisalabad

This is a premium, responsive web template designed for "Cloth Market," a leading fabric business located in Gumti Square, Faisalabad. The website is built to showcase a wide range of high-quality fabrics, including Lawn, Cotton Silk, and Designer Chiffon, with a modern and professional aesthetic.

## 🚀 Features
- **Modern Design:** Elegant aesthetic using a sophisticated color palette (Maroon & Thread-Gold).
- **Responsive Layout:** Optimized for all devices, including mobile phones and tablets.
- **Dynamic Slider:** Integrated video-enhanced hero section for an engaging visual experience.
- **Product Grid:** Clean and organized display of fabric collections.
- **Quick Contact:** Direct CTA buttons for WhatsApp and Phone calls for better client engagement.

## 🛠 Tech Stack
- **HTML5:** Semantic structure for better SEO.
- **CSS3:** Custom styling with responsive design and CSS variables.
- **JavaScript:** Logic for dynamic video slider functionality.
